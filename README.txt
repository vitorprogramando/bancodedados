Sistema banco de dados de cadastrado ligado em xamp com banco de dados em php localhost
para conxão tem que baixar o xamp, ligar o apache e mysql
depois abrir olocal host, criar um banco como crud_db
depois colocar o nome usuarios, e colocar as informações 
id, nome,email, telefone
no mesmo banco de dados tem que colocar o id como primario e marcar a caixinha autentique increment
e depois salvar o banco 
esse mesmo arquivo tem que ficar dentro da pasta htdocs, criar pasta htdocs dentro do local do xamp
e depois colocar cada arquivo com codigo devido no local
e assim conseguimos efetuar a conexão do banco de dados
